# Sound Files

This directory should contain the following audio files:

- `lever.mp3` - Short lever pull sound (0.3-0.6 seconds)
- `spin.mp3` - Loopable spinning/slot machine sound (10-20 seconds)
- `win.mp3` - Victory fanfare sound (1-2 seconds)
- `confetti.mp3` - Short confetti pop sound (optional, 0.2-0.5 seconds)

## Recommended Sources

You can find royalty-free sounds at:
- [Freesound.org](https://freesound.org)
- [Zapsplat.com](https://www.zapsplat.com)
- [Mixkit.co](https://mixkit.co/free-sound-effects/)

## File Format

All files should be in MP3 format with reasonable compression (128-192 kbps is sufficient).

## Volume Levels

Please ensure all audio files have normalized volume levels to avoid jarring transitions.
